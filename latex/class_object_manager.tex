\doxysubsection{Detailed Description}
Manages game objects, including creation, storage, and deletion. 

The \doxylink{class_object_manager}{Object\+Manager} class is responsible for handling all game objects within the simulation. It provides methods to add, remove, and retrieve objects by their names, as well as to access the complete list of objects.
